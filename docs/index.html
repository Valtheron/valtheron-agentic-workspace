Alles in einem Dashboard

Von der Agent-Verwaltung bis zum Notfall-Kill-Switch — alle Werkzeuge für autonome KI-Operationen an einem Ort.

🤖

Agent-Management

290 vordefinierte Agenten in 10 Kategorien. Vollständiges CRUD, Status-Tracking und Persönlichkeitsprofile.

📋

Kanban-Board

Task-Management mit 5 Spalten: Backlog → In Progress → Review → Done → Archiviert. Abhängigkeiten und Prioritäten inklusive.

💬

Echtzeit-Chat

Direktkommunikation mit Agenten via Anthropic Claude, OpenAI GPT oder lokalem Ollama. Persönlichkeitsbasierte Antworten.

📊

Analytics-Dashboard

6 Metriken-Tabs: Trends, Throughput, Fehler, Kapazität, SLA und Erfolgsrate. 7-Tage-Trends mit CSV/JSON-Export.

🔴

Kill-Switch

Sofortstopp aller laufenden Operationen — manuell oder automatisch durch die Rules Engine ausgelöst. 30-Sekunden-Polling.

🔐

Sicherheit & Compliance

AES-256-GCM Vault, TOTP-MFA, JWT+RBAC, Rate Limiting und vollständiger Audit-Trail mit CSV-Export. OWASP Top 10 geprüft.

In 3 Minuten starten

Mit Docker Compose ist die Plattform sofort einsatzbereit.

# Repository klonen
git clone https://github.com/Valtheron/valtheron-agentic-workspace.git
cd valtheron-agentic-workspace

# Umgebungsvariablen konfigurieren
cp backend/.env.example backend/.env

# Starten
docker-compose up -d

# Dashboard öffnen → http://localhost:8080
