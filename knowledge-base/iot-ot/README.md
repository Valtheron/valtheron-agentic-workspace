## Technologiebereiche

### IoT Security
- Firmware-Extraktion und -Analyse
- UART/JTAG Hardware-Debugging
- Kommunikationsprotokolle (MQTT, CoAP, Zigbee)
- RF/SDR Hacking

### OT / ICS / SCADA Security
- Modbus, DNP3, PROFINET Protokolle
- Siemens S7, Allen-Bradley PLCs
- Purdue-Modell und Netzwerksegmentierung
- ICS-spezifische Schwachstellen (Stuxnet-Klasse)

## Wichtige Tools

- **Binwalk** — Firmware-Extraktion
- **Firmwalker** — Firmware-Analyse
- **GNURadio** — SDR/RF-Analyse
- **Shodan** — IoT Device Discovery
- **Claroty / Dragos** — OT Security Monitoring
- **OpenPLC** — PLC Testumgebung

## Normen und Frameworks

- **NERC CIP** — Kritische Infrastruktur (Energie)
- **NIST SP 800-82** — ICS Security Guide
