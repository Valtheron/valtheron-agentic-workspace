## Übersicht

CompTIA bietet eine Progression von Entry-Level (Security+) bis Expert (CASP+), plus spezifische Pfade für Blue Team (CySA+) und Pentesting (PenTest+). Diese Kategorie enthält auch Career-Guides und Interview-Vorbereitung.

## CompTIA Certification Track

```
Security+ (Entry) → CySA+ (Blue) / PenTest+ (Red) → CASP+ (Expert)
```

## Security+ (SY0-701)

**Level:** Entry-Level | **Fragen:** 90 | **Dauer:** 90 Min | **Bestehen:** 750/900

### Prüfungsdomänen (2024)
| Domain | Gewichtung |
|--------|-----------|
| General Security Concepts | 12% |
| Threats, Vulnerabilities & Mitigations | 22% |
| Security Architecture | 18% |
| Security Operations | 28% |
| Security Program Management | 20% |

### Kernthemen
- CIA-Triade, Zero Trust, Defense in Depth
- Authentifizierungsprotokolle (SAML, OAuth, MFA)
- Verschlüsselung (AES, RSA, ECC, PKI)
- Netzwerksicherheit (Firewall, IDS/IPS, VPN)
- Cloud-Sicherheitsmodelle (IaaS, PaaS, SaaS)
- Incident Response, Forensics Basics

## CySA+ (CS0-003)

**Level:** Intermediate | **Fokus:** Blue Team / SOC

- Threat Intelligence und Threat Hunting
- Vulnerability Management
- Incident Response
- Security Operations (SIEM, SOAR)
- Compliance und Reporting

## CASP+ (CAS-004)

**Level:** Expert | **Fokus:** Security Architecture

- Risikoanalyse und -management
- Enterprise Security Architecture
- Cloud/Hybrid Security Design
- Advanced Cryptography
- Research, Development, Collaboration

## Career-Pfade in der Cybersecurity

### Entry-Level (0–2 Jahre)
- **Positionen:** SOC Analyst Tier 1, IT Security Analyst, Junior Pentester
- **Gehalt (DE):** €35.000–50.000
- **Empfohlen:** Security+, eJPT, CySA+

### Mid-Level (3–5 Jahre)
- **Positionen:** Senior SOC Analyst, Penetration Tester, IR Analyst
- **Gehalt (DE):** €55.000–75.000
- **Empfohlen:** OSCP, CySA+, PenTest+

### Senior (5+ Jahre)
- **Positionen:** Red Team Lead, Security Architect, CISO
- **Gehalt (DE):** €80.000–120.000+
- **Empfohlen:** CASP+, CISSP, CISM

## Interview-Vorbereitung

**Typische Fragen:**
- "Was ist der Unterschied zwischen IDS und IPS?"

- "Wie gehst du mit einem Ransomware-Vorfall um?"
- "Was ist SQL Injection? Demonstriere einen einfachen Test"
